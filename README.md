# Romanya-Enerji-Profili-6-Yillik-retim-ve-Tuketim-Verisi
Çalışma, Romanya’nın Mart 2019–Mart 2025 saatlik elektrik üretim-tüketim verisini inceler. Nükleer, rüzgar, hidro, kömür, petrol/gaz, güneş ve biyokütle payları, net enerji dengesi, mevsimsel desenler ve anomaliler analiz edilmiştir. Prophet, 6 aylık tüketim tahmininde en düşük hatayı vermiştir.
